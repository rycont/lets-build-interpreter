# 직접 만들어보는 "yaksok.ts"

[약속 프로그래밍 언어](http://yaksok.org/)의 타입스크립트 구현체인 [yaksok.ts](https://github.com/rycont/yaksok.ts)를 직접 만들어보는 튜토리얼입니다.

## 알아갈 수 있는 내용

이 튜토리얼을 통해서 프로그래밍 언어의 인터프리터를 만드는 방법에 대해 알아가실 수 있길 바랍니다.

## 알고 있어야 할 내용

- 자바스크립트
- 객체지향 프로그램을 작성하는 방법
- 인터프리터와 컴파일러의 차이
